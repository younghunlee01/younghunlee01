# 01. 구성도
[← 목차로 돌아가기](../README.md)

본 프로젝트의 네트워크 구성도입니다. 논리적 구성도(IP / IGP)와 물리적 구성도로 나누어 정리했습니다.

전체 주소 체계는 Class B `YY.YY.X.0/16` 기반이며(YY = Rack 번호, X = 라우터 번호), 아래 구성도 및 설정은 Rack `14` 기준(`14.14.X.X`)으로 작성되었습니다.

---

## 1. 논리적 구성도

### 1-1. IP 구성도

라우터 6대(R1~R6), 스위치 4대(SW1~SW4), 백본 3대(BB1~BB3)로 이루어진 토폴로지이며, WAN 구간은 Frame Relay로 연결됩니다.

![논리적 IP 구성도](images/logical-ip.jpg)

**주요 세그먼트**

| 구분 | 네트워크 | 비고 |
|------|----------|------|
| VLAN_A (#21) | `14.14.20.0/24` | R2 Giga0/1 |
| VLAN_B (#22) | `14.14.62.0/24` | R6 ↔ SW2 트렁크 |
| VLAN_C (#23) | `14.14.55.0/24` | R5 F0/1 |
| VLAN_SWITCHES (#100) | `14.14.90.0/24` | SW1~SW4 SVI |
| VLAN_BB1 (#11) | `150.1.14.0/24` | BB1 `.254` |
| VLAN_BB2 (#12) | `150.2.14.0/24` | BB2 `.254` |
| VLAN_BB3 (#13) | `150.3.14.0/24` | BB3 `.254` |
| Frame Relay | `14.14.13.0/24` | R1 ↔ R3 |
| Frame Relay | `14.14.26.0/24` | R2 ↔ R6 |
| Frame Relay | `14.14.245.0/24` | R2 ↔ R4 ↔ R5 |
| Loopback0 | `14.14.X.X/24` | X = 라우터 번호 |

### 1-2. IGP 구성도

라우팅 도메인을 구간별로 RIPv2 / OSPF(멀티 에어리어) / EIGRP로 분리하고, 경계 지점에서 재분배(Redistribution)로 연동합니다.

![논리적 IGP 구성도](images/logical-igp.jpg)

**라우팅 도메인 구성**

| 프로토콜 | 구간 | 비고 |
|----------|------|------|
| RIPv2 | R1 ↔ R3 (BB1) | `no auto-summary` |
| OSPF Area 0 | R2·R4·R5 (FR), SW1·R6 | 백본 에어리어 |
| OSPF Area 3 | R3 ↔ SW1 | |
| OSPF Area 4 | R2 (Stub, no-summary) | Totally Stubby |
| OSPF Area 5 | R5 (Stub) | |
| OSPF Area 26 | R2 ↔ R6 | **Virtual-Link**으로 Area 0 연결 |
| EIGRP YY (AS 14) | SW1~SW4, R6 | 스위치 백본 |
| EIGRP 100 | R6 (BB3) | `eigrp stub redistribute` |

**재분배 지점**

- RIP ↔ OSPF : **R3**
- OSPF ↔ EIGRP YY : **SW1**
- OSPF ↔ EIGRP 100 : **R6**

## 2. 물리적 구성도

라우터의 Serial 인터페이스는 Frame Relay 클라우드로, Ethernet 인터페이스(e0/0, e0/1)는 스위치(SW1~SW4)로 연결됩니다. 스위치 간은 다중 링크로 묶여 EtherChannel 및 STP로 구성됩니다.

![물리적 구성도](images/physical.jpg)

**연결 개요**

- **WAN** : R1~R6의 Serial 포트 → Frame Relay 클라우드(DLCI 기반 PVC)
- **LAN** : 각 라우터 e0/0·e0/1 → SW1~SW4 (cross-connect)
- **스위치 간 링크** : SW1~SW4 상호 다중 링크 → EtherChannel(Po12/Po13/Po24/Po34)
- **백본** : BB1→SW1, BB2→SW2·SW3, BB3→R6
